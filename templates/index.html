What are Custom Categories?

Custom categories allow you to pass your own category list for the API to match businesses against. If you provide custom categories, the API will prioritize these categories instead of using the default ones.

For example, if you're only interested in matching businesses as "Startup," "Enterprise," or "Non-profit," you can pass these categories, and the API will attempt to classify businesses based on your input.

API Documentation

The Categorme API enables you to categorize businesses, extract contact information, retrieve website metadata, and gain insights from social media and review links.

1. Authorization

To use the API, you must include your unique API Key in the request headers. The API Key ensures secure access and is required for every request. 

  Headers:
    Content-Type: application/json
    X-API-Key:

2. Making a Request

To categorize a business, send a POST request to the following endpoint: 

  POST https://api.categorme.com/api/categorize

Include the following fields in the JSON body of your request:

  • input_string (required): The email or domain/URL to categorize (e.g., "devin@trykeep.com" or "trykeep.com").
  • fields (optional): A list of specific fields to include in the response. If omitted, all fields are returned.
  • custom_categories (optional): An array of categories to override the default category list.

3. Request Example

  Headers:
    Content-Type: application/json
    X-API-Key: 
  
  Body:
  {
    "input_string": "devin@trykeep.com",
    "fields": ["type", "category", "summary", "social_media_links", "review_links", "metadata"],
    "custom_categories": ["Finance", "Retail", "E-commerce"]
  }

4. Response Structure

The API returns a JSON object containing the requested data. Below is an example response: 

  {
    "type": "email",
    "url": "https://trykeep.com",
    "phone_numbers": ["+1-800-555-1234"],
    "emails": ["info@trykeep.com"],
    "website_title": "Keep Business Solutions",
    "website_description": "Helping businesses grow with smart financial tools.",
    "category": "Finance",
    "summary": "Keep offers financial tools for small businesses to manage their expenses and credit efficiently.",
    "social_media_links": [
      {"platform": "Facebook", "url": "https://facebook.com/keepbusiness"},
      {"platform": "Twitter", "url": "https://twitter.com/keepbusiness"}
    ],
    "review_links": [
      {"platform": "Yelp", "url": "https://yelp.com/biz/keep-business"},
      {"platform": "Google Reviews", "url": "https://g.page/r/keep-business/review"}
    ],
    "metadata": {
      "creation_date": "2020-03-15",
      "last_updated_date": "2023-07-10",
      "registrar": "Namecheap, Inc."
    },
    "category_confidence_score": 0.92
  }

5. Additional Features

The API also retrieves the following details:

  • Website Metadata: Information like domain creation date, last updated date, and registrar.
  • Social Media Links: Links to platforms such as Facebook, Twitter, Instagram, and LinkedIn.
  • Review Links: Links to review platforms like Yelp, Google Reviews, or Trustpilot.

6. Fields Filtering

Use the fields parameter to filter the response and include only the desired fields. For example: 

  "fields": ["category", "summary", "social_media_links", "review_links", "metadata"]

This will return only the category, summary, social media links, review links, and metadata fields in the response.

7. Error Handling

If an error occurs, the API returns a JSON object with an error field and an appropriate message. 

  Response Example:
  {
    "error": "Unauthorized or missing X-API-Key"
  }

8. Metadata Details

When metadata is selected in the fields, the following details are provided:

  • Creation Date: The date the domain was first registered.
  • Last Updated Date: The most recent update to the domain registration.
  • Registrar: The company responsible for registering the domain.

9. Supported Platforms for Links

  • Social Media: Facebook, Twitter, Instagram, LinkedIn, YouTube.
  • Review Platforms: Yelp, Google Reviews, Trustpilot, BBB.
