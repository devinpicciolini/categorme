Business Categorizer API

Quickly identify and categorize businesses by simply providing an email or a domain. Our AI-driven categorizer also summarizes the product or service, giving you immediate insights.

Enter your Bearer Token and the email/URL you wish to categorize below, then click “Categorize” to see the results in real-time.

About This API

The Business Categorizer API uses powerful natural language processing to identify, classify, and summarize businesses based on their digital presence. It can parse homepage data, retrieve phone numbers and emails, and provide an at-a-glance understanding of what a company offers.

Built with Flask and integrated with OpenAI's GPT-based services, it demonstrates how easy it is to wrap powerful AI functionality in a secure, scalable API.

Docs

Below is an example of how to use the API. You’ll need your Bearer Token to authenticate. Then simply send a POST request to the following URL: 

POST https://api.bizcategorizer.com/api/categorize
Headers:
  Content-Type: application/json
  Authorization: Bearer <YOUR_TOKEN>

Body:
{
  "input_string": "devin@trykeep.com"
}

Response:
{
  "business_data": {
    "category": "Finance",
    "summary": "...",
    "phone_numbers": [...],
    "emails": [...],
    "website_title": "...",
    "website_description": "...",
    "url": "https://..."
  }
}

If you’re using a local Flask server, replace https://api.bizcategorizer.com with http://localhost:5000 or whatever domain you have set up for your development environment.
